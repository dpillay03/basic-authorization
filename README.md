# Basic Authorization
## Developer: Daniel Pillay
## Description
An exercise that creates basic auth middleware and tests basic signup/signin routes

## Install
1.Clone repository
1.Run `npm install`
1.Run `nodemon` or `node index.js` if nodemon is not installed globally.

## Technologies
1. Language: JavaScript
1. Packages:
- `express`
- `bcrypt`
- `mongoose`
- `superagent`
